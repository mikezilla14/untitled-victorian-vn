# Canon narrative files

This folder contains promoted narrative truth that downstream drafting and review agents may cite.
Do not change canon casually; use the `canon-update` pipeline when content changes affect locked
character, location, or lore facts.

## Validation path

Canon-affecting changes should be reviewed by the lead narrative editor, forensic psychology
consultant when character state is affected, and Victorian consultant when historical setting is
affected. See [../../AGENTS.md](../../AGENTS.md).
