# Continuity Handoff — Release 1 MVP

> **Purpose:** Context-optimized prose continuity for the writers' room. Supplements `story_board.md` (spine/flags/mechanics); does **not** replace canon or voice guides.
>
> **Consumers:** Divergent pool + convergent writer on day `dd` load **only** the section `## Handoff → Day [dd]`.
>
> **Owner:** `convergent_writer` updates the **next** day's section when `dayrdd_non_canon.rpy` passes narrative + historical gates (or on revision pass if exit state changes). Orchestrator may patch intake-only fixes.
>
> **Budget:** ~400–800 tokens per section. No label inventories, no router tables, no full scene summaries.

---

## Template (copy for new days)

```markdown
## Handoff → Day [dd]

**Exit state (Day [dd-1]):** 2–4 sentences — sensory/emotional image the player carries in.

**Flags that matter today:** Bullets only — flags this day must branch on or echo.

**Relationships:** One line each — Stern, Missy, Gideon, Vance (skip if absent).

**Voice today:** Cora + key NPC register (cite voice guide day band).

**Callbacks required:** 3–5 concrete images, phrases, or debts to honor.

**Forbidden:** What not to re-litigate today.

**Entry:** First label + what the opening beat must assume.
```
