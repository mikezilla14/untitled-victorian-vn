# Ren'Py GUI assets

This folder contains Ren'Py GUI image assets and framework-adjacent UI resources for the non-production project.

## Rules

- Treat this as runtime GUI support, not general art scratch space.
- Do not make broad GUI changes as part of a prose or narrative pass.
- If a UI change affects screens/HUD/player flow, route it through a technical implementation or promotion-prep task and validate in the sandbox.

## Related files

- `../screens.rpy` owns screen definitions.
- `../gui.rpy` owns Ren'Py GUI configuration values.
- `../images/ui/` owns project-specific UI art such as overlays and manuscript assets.
