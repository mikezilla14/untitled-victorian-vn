# UI image assets

This folder contains runtime UI images for the non-production Ren'Py project: overlays, manuscript UI, badges, paper textures, sidebars, suspicion vignettes, and related interface art.

## Rules

- Runtime-ready UI assets belong here.
- Prompt exports and source work belong outside this folder unless the game loads them directly.
- Missing structural UI assets should be tracked through the MVP systems integration checklist and the asset manifest process.
- Keep filenames stable once referenced by scripts/screens.
