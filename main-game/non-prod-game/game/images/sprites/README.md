# Sprite assets

Character and UI sprite images for the non-production Ren'Py project live here.

## Rules

- Use stable sprite tags that match existing script conventions, for example `cora_sprite`.
- Keep source exports, prompt logs, and intermediate files outside this runtime folder unless they are deliberately loaded by Ren'Py.
- Manifest-facing names should be predictable and consistent with `docs/art/ASSET_NAMING.md`.
- Scene direction places sprites; it does not create, rename, or retouch image files.
