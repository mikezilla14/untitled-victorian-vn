# Approved & Formatted Assets

This directory houses verified, cleaned, sliced, and fully compressed pre-production assets.

## Deployment Conventions
- All assets here must be formatted as `.webp` (or transparent `.png` where required).
- Naming must strictly follow [docs/art/ASSET_NAMING.md](../../docs/art/ASSET_NAMING.md) rules.
- Once approved, run the copy-to-engine command or manually copy them under `main-game/prod-game/game/images/` for deployment.
