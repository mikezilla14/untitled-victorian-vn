# Raw VNCCS Grid Sprite Sheets

Store uncropped, original VNCCS sprite grid sheets categorized by character folders.

## Folder Conventions
```text
vnccs_sheets/
├── cora/
├── gideon/
├── stern/
├── vance/
└── missy/
```

## Backlog Preservation
Always preserve original VNCCS rendering sheets along with their exact prompt settings. This guarantees that future layered/paper-doll sprite backlogs can be cleanly extracted and matched against the exact lighting models, head angles, and poses of the initial releases.
