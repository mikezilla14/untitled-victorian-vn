# Source Prompts & Text Logs

Save raw image prompt logs, visual reference links, and text prompt experiments here.

## Guidelines
- Keep prompt logs organized by character name or location name.
- Standard file format is text (`*.txt`) or markdown (`*.md`).
- File naming should follow the convention: `{asset_id}_prompts.txt` (e.g. `cora_maid_prompts.txt`).
